###1. npm 명령어   
```
npm prune
```
package.json에 정의되어 있지 않은 불필요 모듈 삭제
```
npm ls -g -depth = 0
```
현재 설치된 전역 모듈 확인
```
npm audit fix
```
npm audit fix 는 취약점을 가진 종속성에 대해 호환되는 버전으로 자동으로 업데이트를 해줍니다.
